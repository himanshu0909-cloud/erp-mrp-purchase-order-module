# ERP / MRP Purchase Order Module

This project is a simple ERP/MRP Purchase Order module built using **FastAPI (backend)** and **HTML, CSS, JavaScript (frontend)**.

It allows users to manage vendors, products, and create purchase orders.

---

## Features

- Add and view vendors
- Add and view products
- Create purchase orders
- Select vendor and multiple products
- Dynamic product rows
- Automatic total price calculation
- Backend stock validation

---

## Tech Stack

Backend
- Python
- FastAPI
- SQLAlchemy
- SQLite

Frontend
- HTML
- CSS
- JavaScript
- Bootstrap
